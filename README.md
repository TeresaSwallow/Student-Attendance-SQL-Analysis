# Student-Attendance-SQL-Analysis
Used PostgreSQL, pgAdmin4 and SQL to analyze synthetic student attendance data, applying joins, filtering, aggregation, and reporting to answer operational questions.

## PROJECT OVERVIEW

Analyze a synthetic student attendance dataset using SQL to answer common operational reporting questions related to attendance, absences, grade levels, and schools.

---

## DATASET

Synthetic student attendance dataset created for portfolio purposes.

Tables:

* Students
* Attendance

---

## TOOLS

* PostgreSQL
* pgAdmin 4
* SQL
* Excel

---

## PROCESS

* Created a PostgreSQL database using pgAdmin 4.
* Imported CSV files into relational database tables.
* Queried attendance records using SQL.
* Applied filtering, aggregation, sorting, and table joins to answer questions.
* Exported query results to Excel for documentation.

---

## ANALYSIS QUESTIONS

1. How many attendance records exist?
2. How many absences occurred?
3. Which grade levels have the most absences?
4. Which schools have the highest attendance?
5. How many attendance records exist for each attendance status?

---

## SQL SKILLS DEMONSTRATED

* SELECT
* WHERE
* COUNT()
* GROUP BY
* ORDER BY
* LEFT JOIN

---

## LIMITATIONS

* Dataset was synthetically generated for educational purposes.
* Analysis focused on descriptive reporting rather than predictive analytics.
* Attendance trends were evaluated using a limited sample dataset.
